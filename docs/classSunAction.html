Detailed Description

An action which causes SunBase to perform an action when used on one.

SunAction is a class that defines actions that can be sent to SunBase and its derivatives. The action string determines which action will be picked from the list of actions in a SunBase. It contains a map of parameters, which can be used to pass along information, and a recursive boolean, which applies only to SunNode, but it makes the node pass the action on to its subnodes.
