Detailed Description

An abstract class that is used to provide services to all SunBase objects.

SunService is an abstract class that is meant to be subclassed and used to provide services to SunBase objects. SunBase has a static map of SunServices, which can be used to identify and use services. SunGame also updates each service every game cycle.
